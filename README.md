## qss的运用

图片中三部分代表这中控件的三种状态，<br>
从左到右分别是：控件Disable，控件未选中，控件上面圆圈选中。<br>
另外控件选中后，颜色也就改为填充白色,白色填充也是动态上下变化的，三角也会随着点击圆圈上下移动<br>
还有没有选中时左侧+-是消失的
